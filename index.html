sticky

Positions:-static, absolute, relative,fixed

Position Static
it is the default position even if apply top ,bottom ,left, right, we cannot see any changes beacuse our position is going to be static (it is not going to move)
position relative:-
it will move according to the browser for first elements , but for the second elements it will move according to the first elements .(it always related to the next element position )
position absolute:-
it will always move according to the parent elements .if the parent elements is not present then it will go for root elements .

position fixed:-it will fixed at specific position .
